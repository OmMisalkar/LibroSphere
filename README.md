# 📚 LibroSphere

## 📌 Description
LibroSphere is a Django-based bookstore web app with buying and borrowing features. It includes user authentication, cart system, and an analytics dashboard using Chart.js to track sales, borrowing trends, and user activity.

## 🚀 Features
- User Authentication
- Book Purchase & Borrow
- Cart System
- Favorites & Reviews
- Analytics Dashboard

## 📊 Dashboard
- Sold vs Borrowed Chart
- Purchase Trend
- Borrow Trend
- Book Ranking

## 🛠 Tech Stack
- Django
- Python
- HTML, CSS, JavaScript
- Chart.js
